# BME 230A — Computational Genomics

Graduate-level computational genomics coursework at UC Santa Cruz. Covers single-cell RNA-seq analysis, doublet detection, count distribution modeling, and RNA distance metrics applied to real GEO datasets.

## Notebooks

| File | Description |
|------|-------------|
| `count_distribution_modeling.ipynb` | Model count distributions in scRNA-seq data |
| `doublet_detection_scrna_seq.ipynb` | Detect doublets in single-cell libraries |
| `rna_modeling_distance_metrics.ipynb` | RNA expression distance metrics and dimensionality reduction |

## Dataset

- `dataset_GSE186078/` — Single-cell RNA-seq dataset from GEO (GSE186078), anndata format

## Skills

Single-cell RNA-seq analysis · Doublet detection · Count distribution modeling · Dimensionality reduction · Python · scanpy · anndata · Jupyter · HDF5
